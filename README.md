# AI x Web3 School Learning Repo

Personal learning journal and proof-of-work for AI x Web3 School.

AI x Web3 School is a learning program for exploring how AI agents, Web3 infrastructure, wallets, governance, security, and programmable digital economies can be combined in practical builder workflows.

My learning focus is Programmable Compliance / 可编程合规: treating compliance as an upfront product and protocol design layer, not merely an after-the-fact legal remedy.

## Privacy Rule

This repo is designed to be public proof-of-work. Do not commit API keys, wallet seed phrases, private keys, passwords, unpublished contact info, private meeting links, internal documents, or other people's personal data.

## Structure

- `profile.md`: learner profile and preferences.
- `learning-plan.md`: personal learning plan and weekly rhythm.
- `daily/`: daily notes and check-in drafts.
- `tasks/`: course tasks and deliverables.
- `experiments/`: small prototypes and technical experiments.
- `handbook-feedback/`: feedback, questions, unclear concepts, outdated references, and improvement suggestions for the Handbook.
- `hackathon/`: project ideas, design notes, and demo assets.
- `submissions/`: submitted links and records.
- `templates/`: reusable daily and task templates.
